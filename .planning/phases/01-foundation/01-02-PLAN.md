AcroForm field structure (forms/f1041.pdf — confirmed 184 fields):

Page layout:
  Page1[0]: 60 text + 27 checkbox fields
    - DateNameAddress_ReadOrder[0]: f1_1[0] through f1_11[0] (trust name, EIN, dates, address)
    - Checkboxes c1_1[0] through c1_9[0] (Lines A-B type-of-entity checkboxes)
    - Income lines: f1_15[0] onward (Line 1 interest, Line 2a ordinary dividends, etc.)
    - Deduction lines, exemption, taxable income, tax
  Page2[0]: 42 text + 0 checkbox fields
    - Schedule A (charitable deduction)
    - Schedule B (income distribution deduction — zero for accumulation trust)
    - Schedule D summary lines
  Page3[0]: 14 text + 30 checkbox fields
    - Schedule G Part I (tax computation) + Part II (credits/recapture)
    - Other Information Yes/No boxes (c3_*)

Field naming convention:
  Full hierarchical key (required by pypdf): topmostSubform[0].Page{N}[0].{section}[0].{id}[0]
  Text fields: f{page}_{seq}[0] — e.g., f1_15[0], f2_23[0]
  Checkboxes: c{page}_{seq}[0] — e.g., c1_1[0], c3_1[0]
  On-state values for checkboxes in this PDF: '1' or '2' (NOT 'Yes')

Mapping method (per locked decisions + research pitfall #4):
  The filed return (filed_returns/2024 1041...) has NO AcroForm fields — it is flattened/scanned.
  Do NOT attempt to extract values from it programmatically.
  Instead:
    1. Run get_fields() against forms/f1041.pdf to enumerate all field IDs with types
    2. Run extract_text() on forms/f1041.pdf to read the printed line labels in order
    3. Match line labels to field IDs by their sequential position on the page
    4. Use the filed return PDF visually (open it) to confirm which lines had values in 2024

  The 2024 reference return has these key lines populated (from ROADMAP success criteria):
    - Interest income (Line 1)
    - Ordinary dividends (Line 2a)
    - Qualified dividends (Line 2b — checkbox line, not a dollar amount on page 1)
    - Net short-term capital gain from Schedule D (Line 3)
    - Net long-term capital gain from Schedule D (Line 4)
    - Other income (if any)
    - Total income
    - Deductions (trustee fees, etc. if present)
    - Exemption ($100)
    - Taxable income
    - Schedule D: short-term gain/loss, long-term gain/loss summary lines
    - Schedule G: tax on taxable income, NIIT amount
    - Form 8960: NII components, NIIT

      if ft == '/Btn':
