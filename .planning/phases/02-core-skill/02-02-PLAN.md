**compute_schedule_d(transactions):**
For each transaction: adjusted_cost = cost + wash_sale (add wash sale disallowed back per IRS Form 8949 Box 1g). gain_loss = proceeds - adjusted_cost.
Separate into short-term (form8949_code == 'A') and long-term (form8949_code == 'D').
Return:
```python
{
    'st_proceeds': float, 'st_cost': float, 'st_wash_sale': float, 'st_net': float,
    'lt_proceeds': float, 'lt_cost': float, 'lt_wash_sale': float, 'lt_net': float,
    'net_combined': float,
    'rows': list of transaction dicts (original + gain_loss key added)
}
```

**compute_form_1041_page1(csv_data, sched_d, cfg):**
Returns dict of all page 1 computed values:
```python
{
    'interest_income': csv_data['int_box1'],
    'ordinary_dividends': csv_data['div_1a'],
    'qualified_dividends': csv_data['div_1b'],  # all to estate/trust (non-distributing)
    'capital_gain_loss': sched_d['net_combined'],
    'total_income': sum of interest + ordinary_dividends + capital_gain_loss,
    'exemption': cfg['trust']['exemption'],
    'taxable_income': total_income - exemption,
}
```
Note: total_income uses ordinary dividends (not qualified dividends — qualified is a subset already included in ordinary). Interest + ordinary_dividends + capital_gain_loss = gross income.

**compute_schedule_b(page1):**
Returns Schedule B field values for a non-distributing trust. All distribution fields are $0.
```python
{
    'adjusted_total_income': page1['total_income'],  # flows from Page 1 Line 17
    'income_required_to_be_distributed': 0.0,
    'other_amounts_distributed': 0.0,
    'total_distributions': 0.0,
    'distributable_net_income': page1['total_income'],
    'income_distribution_deduction': 0.0,
}
```

**compute_schedule_g(taxable_income, qual_div, lt_net_gain, cfg):**

```python
    'tax_on_taxable_income': float,  # Line 1a = ordinary_tax + cap_gains_tax
    'total_tax': float,              # Line 9 = tax_on_taxable_income (no other additions for this trust)
}
```

**compute_form_8960(csv_data, page1, cfg):**
Implement as documented in 02-RESEARCH.md Pattern and code examples.

```python
{
    'interest_income': csv_data['int_box1'],
    'ordinary_dividends': csv_data['div_1a'],
    'net_gain_from_dispositions': sched_d_net_combined,  # pass as parameter
    'total_nii': interest + ordinary_dividends + net_gain,  # Line 8
    'agi_undistributed_nii': page1['taxable_income'],       # Line 9b
```
