description: "Fill Form 1041 and all required attachments from brokerage CSV"
---

Fill Form 1041 and all required attachments (Schedule D, Form 8960, Form 8949) from the brokerage year-end CSV. Parses the CSV, runs all tax computations, validates, and writes filled PDFs to the output/ directory.

Note: The `cd` command is required because fill_1041.py uses relative paths (config/, forms/, output/) and must run from the project root.

```bash
cd /c/Users/TrentFoley/Source/dynasty-trust-taxes && /c/ProgramData/miniconda3/python fill_1041.py $ARGUMENTS
```

Supported flags:
- `--dry-run`: print all computed values without writing any files
- `--year YYYY`: use config/YYYY.json (default: 2025)
- `--csv PATH`: override CSV input path
- `--output-dir PATH`: override output directory
